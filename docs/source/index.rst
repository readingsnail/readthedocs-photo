사진 기술 혹은 이론, 여담
===================================

이 사이트는 **사진관련 기술 이나 여담**등을 이야기 하는 장소입니다.

수정을 원하시면 첫번째 방법으로는 `리누스 토발즈라는 욕쟁이가 빡쳐서 수주안에 만든 물건 <https://github.com/readingsnail/readthedocs-photo/>` 혹은 당근이나 기타 등등으로 알려주세요

내용은 수시로 변경될수 있습니다. 잘 알려주세요.
